asteroid.losses.cluster.deep_clustering_loss(embedding, tgt_index, spk_cnt=None)[source]¶

Compute the deep clustering loss defined in :param embedding: torch.Tensor. Expected shape [batch, frame x frequency, embeddingDim] :param tgt_index: torch.LongTensor. Dominating source in each time frequency bin. Expected shape:

[batch, frequency, frame]

Parameters

spk_cnt – int. Number of speakers. Estimated from tgt_index if not given.

Returns

torch.Tensor. Deep clustering loss for every sample

Examples

>>> import torch
>>> from asteroid.losses.cluster import deep_clustering_loss
>>> spk_cnt = 3
>>> embedding = torch.randn(10, 5*400, 20)
>>> targets = torch.LongTensor(10, 400, 5).random_(0, spk_cnt)
>>> loss = deep_clustering_loss(embedding, targets, spk_cnt)
>>> print(loss.size)
torch.size([10])
